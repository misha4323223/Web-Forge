# Регламент работы WebStudio
## Полный цикл: от звонка клиента до сдачи проекта

---

## РАСПРЕДЕЛЕНИЕ РОЛЕЙ

| Вы (владелец студии) | Я (ИИ в Replit) |
|---------------------|-----------------|
| Общаетесь с клиентами | Создаю сайты |
| Собираете брифы | Пишу код |
| Согласовываете дизайн | Делаю интеграции |
| Принимаете оплату | Настраиваю хостинг |
| Передаёте мне задачи | Выполняю задачи |

---

## ЭТАП 1: ПЕРВИЧНЫЙ КОНТАКТ

### Ваши действия:
Когда клиент звонит или пишет — задайте эти вопросы:

| № | Вопрос | Зачем |
|---|--------|-------|
| 1 | Как вас зовут? | Обращение |
| 2 | Расскажите о бизнесе в 2-3 предложениях | Понять нишу |
| 3 | Что нужно: лендинг, корпоративный сайт или интернет-магазин? | Тип проекта |
| 4 | Есть ли домен? | Размещение |
| 5 | Когда нужен готовый сайт? | Сроки |
| 6 | Какой бюджет? | Цена |
| 7 | Есть примеры сайтов, которые нравятся? | Стиль |
| 8 | Нужны ли платежи/корзина на сайте? | Интеграции |

### Результат этапа:
- Вы понимаете, что нужно клиенту
- Назначена встреча для подробного брифа

---

## ЭТАП 2: БРИФИНГ

### Ваши действия:
Соберите у клиента следующую информацию:

#### Для лендинга/корпоративного сайта:

**О компании:**
- [ ] Название компании
- [ ] Чем занимаются
- [ ] Главное преимущество (почему выбирают их)
- [ ] Кто их клиенты

**Материалы от клиента:**
- [ ] Логотип (файл)
- [ ] Фотографии (товары, команда, офис)
- [ ] Тексты (или вы напишете сами)
- [ ] Отзывы клиентов
- [ ] Контакты (телефон, email, адрес)

**Пожелания по дизайну:**
- [ ] Какие цвета нравятся?
- [ ] Примеры сайтов (ссылки)
- [ ] Стиль: строгий / яркий / минималистичный

**Функционал:**
- [ ] Форма заявки — куда отправлять? (Email / Telegram / WhatsApp)
- [ ] Нужна ли карта с адресом?
- [ ] Ссылки на соцсети (VK, Telegram)

---

#### Для интернет-магазина (дополнительно):

**Товары:**
- [ ] Сколько товаров?
- [ ] Сколько категорий?
- [ ] Есть ли прайс-лист (Excel)?
- [ ] Фото товаров

**Оплата:**
- [ ] Способы оплаты (карта, СБП, при получении)
- [ ] Есть ли юрлицо? (ИНН, название)

**Доставка:**
- [ ] Какая доставка? (курьер, СДЭК, почта, самовывоз)
- [ ] По какой географии? (город / вся Россия)

**Уведомления:**
- [ ] Куда приходят заказы? (Telegram / Email)

---

### Что вы говорите мне после брифа:

```
Новый проект: [Название]

Тип: Лендинг / Корпоративный сайт / Интернет-магазин

О клиенте:
- Компания: [название]
- Сфера: [чем занимаются]
- УТП: [главное преимущество]

Материалы:
- Логотип: [есть/нет, приложен]
- Фото: [есть/нет, приложены]
- Тексты: [есть/нет]

Дизайн:
- Цвета: [какие]
- Стиль: [какой]
- Примеры: [ссылки на понравившиеся сайты]

Функционал:
- Форма заявки → отправлять в [Telegram/Email]
- Карта: [нужна/не нужна]
- Соцсети: [ссылки]

Для магазина:
- Товаров: [количество]
- Категорий: [количество]
- Оплата: [способы]
- Доставка: [способы]
```

### Мои действия:
- Создаю проект
- Собираю структуру сайта
- Добавляю контент
- Даю вам ссылку на превью

## ЭТАП 3: СОЗДАНИЕ САЙТА

### Ваши действия:
1. Отправляете мне бриф (см. выше)
2. Получаете от меня ссылку на превью
3. Показываете ссылку клиенту
4. Собираете правки от клиента
5. Передаёте правки мне

### Как передать мне правки:

```
Правки от клиента:

1. [Что изменить] — [как должно быть]
2. [Что изменить] — [как должно быть]
3. ...

Или:
- Поменять цвет кнопок на синий
- Добавить ещё один блок с услугами
- Убрать секцию "О нас"
- Поменять фото на главном экране
```

### Мои действия:
- Вношу правки
- Даю обновлённую ссылку
- Повторяем до согласования

## ЭТАП 4: СОГЛАСОВАНИЕ С КЛИЕНТОМ

### Ваши действия:

**Отправьте клиенту ссылку:**
```

Первая версия сайта готова: [ссылка]

— Что нравится?
— Что хотите изменить?

Жду обратной связи!
```

**После правок:**
```
[Имя], добрый день!

Внёс изменения: [ссылка]

Посмотрите, пожалуйста:
— Всё ли учтено?
— Можем финализировать?

### Результат:
- Клиент подтвердил: "Всё нравится, запускаем"
- Вы говорите мне: "Сайт согласован, переходим к размещению"

```

Домен: [название домена или "нет домена"]

Если магазин:

- Telegram бот: [для уведомлений о заказах]

### Мои действия:

- Подключаю домен
- Настраиваю SSL (безопасное соединение)
- Подключаю Яндекс.Метрику
- Подключаю интеграции (ЮКасса, Telegram)
- Даю вам финальную ссылку на рабочем домене

## ВАРИАНТЫ РАЗМЕЩЕНИЯ САЙТОВ

### Вариант 1: Яндекс Cloud (для сложных проектов)

**Когда использовать:**
- Интернет-магазины с базой данных
- Проекты с высокой нагрузкой
- Нужна синхронизация с 1С

**Что размещаем:**
| Компонент | Сервис Яндекс Cloud |
|-----------|---------------------|
| Статика (HTML, CSS, JS) | Object Storage + CDN |
| База данных | Managed PostgreSQL |
| Бэкенд (API) | Cloud Functions или Compute Cloud |
| Файлы (фото товаров) | Object Storage |

**Стоимость:** от 500-1500 ₽/мес (зависит от нагрузки)

**Плюсы:**
- Надёжность и скорость
- Масштабируется при росте
- Всё в одном месте

---

### Вариант 2: Timeweb (простой и дешёвый)

**Когда использовать:**
- Лендинги и корпоративные сайты
- Небольшие интернет-магазины (до 100 товаров)
- Быстрый запуск

**Тарифы:**
| Тариф | Цена | Для чего |
|-------|------|----------|
| Эконом | от 126 ₽/мес | Лендинги |
| Year+ | от 180 ₽/мес | До 5 сайтов |
| Optimo+ | от 226 ₽/мес | Корпоративные сайты, магазины |

**Что входит:**
- Бесплатный SSL (https)
- Домен .ru в подарок при оплате за год
- Резервные копии каждые 6 часов
- 10 дней тестирования бесплатно
- Поддержка 24/7

**Плюсы:**
- Очень дёшево
- Простая панель управления
- Оплата картами РФ и СБП

---

### Какой вариант выбрать?

| Проект | Рекомендация |
|--------|--------------|
| Лендинг | **Timeweb** (от 126 ₽/мес) |
| Корпоративный сайт | **Timeweb** Optimo+ (от 226 ₽/мес) |
