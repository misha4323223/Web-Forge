## ЭТАП 3: ДИЗАЙН И ПРОТОТИП (Дни 3-7)

> **Важно:** Мы делаем дизайн сразу в коде (в Replit), а не в отдельных программах.
> Клиент видит живой сайт с анимациями, а не статичную картинку.

### 3.1 Создание первой версии (2-3 дня)

**Действия:**
1. Создать новый проект в Replit для клиента
2. Собрать базовую структуру страницы
3. Добавить контент клиента (тексты, фото, логотип)
4. Отправить клиенту ссылку на превью

**Как показать клиенту:**
```
1. В Replit нажать кнопку "Run" (запустить проект)
2. Нажать "Open in new tab" — откроется живой сайт
3. Скопировать URL и отправить клиенту
4. Клиент смотрит реальный сайт на своём телефоне/компьютере
```

**Чек-лист первой версии:**
- [ ] Шапка с логотипом и контактами
- [ ] Hero-секция с УТП и CTA
- [ ] Блок преимуществ
- [ ] Услуги/товары
- [ ] О компании
- [ ] Отзывы (если есть)
- [ ] Форма заявки
- [ ] Контакты + карта
- [ ] Футер

### 3.2 Доработка дизайна (2-3 дня)

**Действия:**
1. Настроить цветовую палитру под бренд клиента
2. Подобрать шрифты (Google Fonts)
3. Добавить анимации и эффекты
4. Проверить мобильную версию
5. Отправить клиенту обновлённую ссылку

**Чек-лист дизайна:**
- [ ] Логотип интегрирован
- [ ] Цвета соответствуют брифу
- [ ] Контраст текста достаточный
- [ ] Кнопки заметные
- [ ] Мобильная версия готова
- [ ] Все фото обработаны

### 3.3 Согласование с клиентом

**Отправляем клиенту:**
- Ссылку на живой сайт (превью из Replit)
- Видео-презентацию (запись экрана) — по желанию

**Фиксируем:**
- Правки от клиента (не более 2 итераций включено в стоимость)
- Финальное подтверждение дизайна

## ЭТАП 6: РАЗМЕЩЕНИЕ (Дни 17-18)

### 6.1 Подготовка к размещению

**Чек-лист:**
- [ ] Домен зарегистрирован/передан
- [ ] DNS настроен
- [ ] SSL-сертификат (Let's Encrypt)
- [ ] Яндекс.Метрика подключена
- [ ] Политика конфиденциальности добавлена

### 6.2 Варианты размещения

**Для лендингов (статика):**

| Сервис | Команды |
|--------|---------|
| Яндекс Object Storage | Загрузить build в бакет, включить хостинг |
| Timeweb Cloud | VPS + nginx или S3 |
| Vercel/Netlify | git push → автодеплой |

**Для интернет-магазинов:**

| Компонент | Сервис | Настройка |
|-----------|--------|-----------|
| Frontend | Яндекс Object Storage + CDN | Статика |
| Backend | Яндекс Cloud Functions или VPS | API |
| База данных | Yandex Managed PostgreSQL | |
| Файлы | Яндекс Object Storage | Картинки товаров |

### 6.3 Настройка домена

```
1. Войти в панель регистратора (REG.RU, RU-CENTER)
2. Добавить A-запись → IP сервера
3. Добавить CNAME для www
4. Дождаться обновления DNS (до 24ч)
5. Выпустить SSL через Let's Encrypt
```
